# ОренСпецСервис

Премиальный многостраничный корпоративный сайт для компании «ОренСпецСервис».

## Стек

- HTML
- CSS
- JavaScript

## Основные страницы

- `index.html` — Главная
- `about.html` — О компании
- `ritual-services.html` — Ритуальные услуги
- `transport.html` — Транспорт
- `farewell-halls.html` — Прощальные залы
- `monuments.html` — Памятники
- `catalog.html` — Каталог памятников
- `manufacturing-installation.html` — Изготовление и установка
- `engraving-3d.html` — Гравировка и 3D-макет
- `works.html` — Наши работы
- `prices.html` — Цены
- `faq.html` — FAQ
- `contacts.html` — Контакты
- `privacy.html` — Политика конфиденциальности

## Структура проекта

- `css/style.css` — общие стили
- `js/script.js` — общая интерактивность
- `images/` — изображения сайта
- `video/` — видеофайлы сайта

## Запуск

Сайт статический: достаточно открыть `index.html` в браузере или загрузить проект на GitHub Pages / любой статический хостинг.
